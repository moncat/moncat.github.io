    项目经理管理

    carl 10/21/2023 管理

    # 1.项目经理管理

    # 1.1.宗旨

    首先要做的，就是要尽快树立起自己在团队里的威信。

    威信是管理者实现领导意图、实施有效管理的无形资产和基本条件，然而威信并不是上级能任命的，也不是花钱就能买到的，它必须靠日积月累的努力才能赢得。

    # 1.2.方法论

    管理团队，你首先要做的，就是要尽快树立起自己在团队里的威信。

    威信是管理者实现领导意图、实施有效管理的无形资产和基本条件，然而威信并不是上级能任命的，也不是花钱就能买到的，它必须靠日积月累的努力才能赢得

    # 1.1.1.以德服众

    品行、作风，良好的道德情操是受人敬仰的基本条件。这些看起来比较空泛的软实力却是你成为一个好主管必须的基础。 严于律己、宽以待人，凡是身先士卒，遇事主动担责，时刻体谅，关心队员等这些常规做法，都是你树立个人威信的必行之事

    # 1.1.2.公正公开

    带团队一定要学会“公平，公正，公开”三项基本法则，切不可凭一己之私，随意行使职务权利。 你的做法，说辞已经不仅仅是你的个人意见，而是会影响到别人的。 对待队员务必要一视同仁，不能厚此薄彼，不能分亲疏、拉帮派，要将一碗水端平，按制度、按程序，按规则办事是必须的。并且要常怀律己之心，耐得住诱惑和冷漠，所谓己身不正，何以正人？说的就是这个道理。 当员工询问你的意见，你应该考虑周全，表达有力。不应支吾拖延。做大决定时设定期限。如果中途听取他人意见，觉得应该改变决定，就尽快改变决定，接收新观念。 前后行动一致，要团队成员敢于向你反馈问题。

    # 1.1.3.专业技能

    团队是为完成任务设立的，作为团队管理者，如果专业不到位、说话破绽百出、遇事束手无策的，就无法和开发人员交流，无法抓住问题核心矛盾。 所以，必须得不断学习业务技术，努力优化自身的知识结构和能力结构，增强自己的技术能力，只有在计算机专业本领过硬，才可增加威望。 这里的技术能力，不限于需求调研，文档编写，后端开发，前端开发，测试用例，服务器部署，服务器运维。

    # 1.1.4.业绩说话

    一切理论都要落实到实际开发。没有落实到业绩，都是空谈。 需要以结果为导向，把握好开发进度，测试进度，交付时间，及时给客户展现处一个可以完整使用的项目成果。

    # 1.1.5.以诚取威

    以诚取威，诚就是诚实与守信。 诚实就是实话实说，不欺瞒，守信就是说到做到，不失言，不爽约。 管理者要提高自己的公信力，就要做到为人厚道、做事诚信、表里如一，胸怀坦荡，一诺千金，唯有如此，才能在队员心中树立较高的威望。

    # 1.1.6.通情达理

    作为团队管理者要学会打点感情牌。 团队是人的聚会，所以你、千万不要脱离了你的团队，不要用官职压人，不要摆架子，如果你装腔作势，高高在上，队员就会敬而远之，你也就没有办法进行深入的沟通。自然也就怪不得别人阳奉阴违，各有一套了。 好的主管会让人信服，信服的不是他的那个小小的头衔，而是他就活生生就在你身边，但却比你优秀上那么一层。作为团队的后盾和支持，让团队成员喜欢沟通，习惯沟通。

    # 1.1.7.培养人才

    团队扩大，一定需要更多的团队管理人员。 所以要学会培养跟随者，首先要选出优秀的队员加以培养，考察该队员的承压能力，沟通能力，上进心，责任心。 其次要毫不吝啬的拿出自己最犀利的经验分享给你的队员，记住不要藏私，现在的年轻人都很有眼光，他们分得清那些是在忽悠人，哪些是真的在教他们。 分享经验，不限于队员管理，还有技术分享。将自己手中的一些任务分配下去，信任队员，队员机会。 当然，你得有，才能教他们。如果是新技术，也要自己先下功夫，至少根据经验把握好技术研究的方向。

    # 1.1.8.交流沟通

    对队员要多沟通、多交流、多关心、多支持，多作换位思考，多设身处地地为队员着想，并能及时解决队员在生活和工作中的实际困难，使他们由衷地对你产生出亲近，依赖之情。 假如你刚开始并不会怎样做个好主管，那么在工作之余，你就要学会勤快的自己下功夫学习外，也要勤快的和队员保持紧密的沟通，只有这样你才会得到工作上的相互支持和理解。 保持幽默感，让管理者让人觉得可以亲近，关系和谐融洽。

    # 1.1.9.奖励惩罚

    队员做的好时，要公开奖励。 不一定是实物奖励，但要让他得到团队的荣耀，让他感受到主管的重视，努力有成果，让他有成就感。 要批评队员是，尽量私下进行，根据实际情况，恩威并重。 重要的不是批评，而是要找到问题原因，和解决问题的办法。 要对事不对人，把握分寸。重要不要伤害了对方的自尊心。只有做到这些，你才能赢得队员的尊敬和爱戴。

    # 1.1.10.上传下达

    做好“通道”作用上传下达是这个岗位的职能之一，所以要学会替团队成员在领导面前争取，同时也要学会在团队面前替公司分忧。 不能只做领导的传声筒，也不能有事了就往队员身上推，这其中的分寸拿捏实在是项最考验人项目，它只和你是否懂人情世故有关，只和你是否了解人性有关。 这是对你会不会处世为人的直接检验，也是区分一个主管职级是否合格的标准之一，但其中的尺度该如何拿捏却是无法详细描述，全在真诚服务公司和队员的心，和悟性。
